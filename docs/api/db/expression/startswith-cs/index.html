    startswith_cs (other)

    • Method of Expression

    This evaluates to true if the string :ref on the left side starts with the string on the right side. This method operates in a case-sensitive manner. For the case-insensitive method see startswith :ref.

    Example

    <h1>A</h1>
<ul>
{% for item in this.children.filter(F.name.startswith_cs('A')) %}
  <li>{{ item.name }}
{% endfor %}
</ul>
